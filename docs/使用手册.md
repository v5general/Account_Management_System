## 1. 系统概述

### 1.1 系统简介

账务管理系统是一套轻量化、易操作的企业收支管理工具，支持：

- 收支流水的规范化记录
- 凭证附件的统一管理
- 多维度数据统计分析
- 按角色权限管理

### 1.2 用户角色

| 角色 | 权限范围 |
|------|----------|
| **管理员** | 拥有所有权限，包括用户管理、系统设置、查看全部数据 |
| **财务人员** | 可登记收支、管理费用分类、查看统计报表、查看全部收支记录 |
| **员工** | 仅可查看本人相关的收支记录和凭证 |

### 1.3 功能模块

| 模块 | 管理员 | 财务人员 | 员工 |
|------|--------|----------|------|
| 首页仪表盘 | ✓ | ✓ | ✓ |
| 收入登记 | ✓ | ✓ | - |
| 支出登记 | ✓ | ✓ | - |
| 收支列表 | ✓ | ✓ | ✓（仅个人） |
| 费用分类 | ✓ | ✓ | - |
| 统计报表 | ✓ | ✓ | - |
| 用户管理 | ✓ | - | - |
| 操作日志 | ✓ | - | - |

## 2. 登录与退出

### 2.1 系统登录

**步骤：**

1. 打开浏览器，访问系统地址（如：`http://your-domain.com`）
2. 进入登录页面，输入用户名和密码
3. 点击「登录」按钮

![登录页面示意图]

**默认账号：**

| 用户名 | 密码 | 角色 |
|--------|------|------|
| admin | admin123 | 管理员 |

**注意事项：**
- 密码连续输入错误5次，账号将被锁定30分钟
- 登录令牌有效期为24小时，超时需重新登录

### 2.2 退出登录

**步骤：**

1. 点击页面右上角的用户名
2. 在下拉菜单中选择「退出登录」
3. 确认退出

## 3. 管理员操作指南

### 3.1 首页仪表盘

登录后，首页展示以下信息：

| 数据项 | 说明 |
|--------|------|
| 总收入 | 统计期间内的总收入金额 |
| 总支出 | 统计期间内的总支出金额 |
| 净收支 | 收入与支出的差额 |
| 记录数 | 收支记录的总数量 |
| 最近收支 | 最近10笔收支记录列表 |
| 项目统计 | 按项目统计的收支占比图表 |

### 3.2 用户管理

#### 3.2.1 新增用户

**步骤：**

1. 进入「系统设置」→「用户管理」
2. 点击「新增用户」按钮
3. 填写用户信息：
   - 用户名（必填，唯一）
   - 密码（必填，至少8位）
   - 角色（必填）
   - 部门（选填）
4. 点击「确定」保存

**角色说明：**

| 角色 | 说明 |
|------|------|
| 管理员 | 拥有系统所有权限 |
| 财务人员 | 可进行收支登记、查看报表 |
| 员工 | 仅可查看个人相关记录 |

#### 3.2.2 编辑用户

**步骤：**

1. 在用户列表中找到目标用户
2. 点击「编辑」按钮
3. 修改用户信息（用户名不可修改）
4. 点击「确定」保存

#### 3.2.3 重置密码

**步骤：**

1. 在用户列表中找到目标用户
2. 点击「重置密码」按钮
3. 输入新密码（至少8位）
4. 点击「确定」

**注意：** 重置密码后，用户需使用新密码登录。

#### 3.2.4 启用/禁用用户

**步骤：**

1. 在用户列表中找到目标用户
2. 点击「禁用」或「启用」按钮
3. 确认操作

**说明：** 被禁用的用户无法登录系统。

### 3.3 操作日志

**功能：** 查看系统所有操作记录

**查询条件：**
- 操作类型：登录、创建、更新、删除
- 时间范围

**日志内容：**
- 操作时间
- 操作人
- 操作类型
- 操作模块
- 操作描述
- IP地址
- 操作状态

## 4. 财务人员操作指南

### 4.1 收入登记

**步骤：**

1. 进入「收支管理」→「收入登记」
2. 填写收入信息：
   - **来源项目**（必填）：收入来源的项目名称
   - **费用分类**（选填）：选择收入对应的费用分类
   - **金额**（必填）：填写收入金额，自动显示为正数
   - **交易时间**（必填）：选择收入发生的日期和时间
   - **备注**（选填）：填写相关说明
3. 上传凭证附件：
   - 点击「上传凭证」按钮
   - 选择文件（支持jpg、png、pdf格式）
   - 单个文件不超过100MB
   - 至少上传1个凭证
4. 点击「提交」完成登记

**注意事项：**
- 每笔收入记录必须至少关联1个凭证附件
- 金额自动保留两位小数

### 4.2 支出登记

**步骤：**

1. 进入「收支管理」→「支出登记」
2. 填写支出信息：
   - **费用分类**（必填）：选择支出对应的费用分类
   - **关联项目**（必填）：支出关联的项目名称
   - **关联人员**（必填）：选择支出关联的人员
   - **金额**（必填）：填写支出金额
   - **交易时间**（必填）：选择支出发生的日期和时间
   - **备注**（选填）：填写相关说明
3. 上传凭证附件（至少1个）
4. 点击「提交」完成登记

**注意事项：**
- 支出记录必须关联人员，金额自动显示为负数
- 常见费用分类：工资、设备采购、差旅费、业务招待费等

### 4.3 收支列表查询

**查询条件：**

| 条件 | 说明 |
|------|------|
| 时间范围 | 选择开始和结束日期 |
| 费用分类 | 按费用分类筛选 |
| 项目 | 输入项目名称模糊搜索 |
| 类型 | 收入/支出/全部 |

**列表操作：**
- **查看**：点击查看记录详情
- **凭证**：点击查看关联的凭证附件（显示附件数量）

### 4.4 费用分类管理

#### 4.4.1 新增分类

1. 进入「费用分类」
2. 点击「新增分类」
3. 填写分类信息：
   - 分类名称（必填）
   - 描述（选填）
   - 排序（选填，数字越小越靠前）
4. 点击「确定」

**预设分类：**
- 工资
- 设备采购
- 服务购买
- 差旅费
- 业务招待费
- 办公费用
- 其他

#### 4.4.2 编辑/删除分类

- **编辑**：修改分类名称、描述或排序
- **删除**：删除未被使用的分类

**注意：** 已关联收支记录的分类无法删除。

### 4.5 统计报表

**功能：** 多维度、多周期统计分析

**统计维度：**
- **按项目**：统计各项目的收支情况
- **按人员**：统计各人员的收支情况
- **按分类**：统计各费用分类的收支情况

**统计周期：**
- 月度
- 季度
- 年度

**操作步骤：**

1. 进入「统计报表」
2. 选择统计维度和周期
3. 选择时间范围
4. 点击「查询」

**报表内容：**

| 数据项 | 说明 |
|--------|------|
| 总收入 | 时间范围内的总收入 |
| 总支出 | 时间范围内的总支出 |
| 净收支 | 收入减去支出的差额 |
| 记录数 | 收支记录的总数 |
| 明细数据 | 按维度展示的详细统计 |

**查询条件：**
- 时间范围
- 费用分类
- 项目
- 收支类型

- 费用分类
- 金额

## 6. 常见问题

### 6.1 登录相关

**Q: 忘记密码怎么办？**

A: 请联系管理员重置密码。

**Q: 账号被锁定怎么办？**

A: 连续输错密码5次会导致账号锁定30分钟，请等待后重试，或联系管理员解锁。

**Q: 提示"令牌过期"怎么办？**

A: 令牌有效期为24小时，超时后请重新登录。

### 6.2 收支登记

**Q: 为什么上传文件失败？**

A: 请检查：
- 文件格式是否为 jpg、png 或 pdf
- 文件大小是否超过 100MB
- 网络连接是否正常

**Q: 可以修改已登记的收支记录吗？**

A: 可以，财务人员和管理员可以编辑已登记的收支记录。

**Q: 删除的收支记录还能恢复吗？**

A: 删除操作为软删除（标记为已作废），如需恢复请联系系统管理员。

### 6.3 查询与统计

**Q: 为什么看不到某些收支记录？**

A:
- 员工只能查看本人关联的记录
- 请确认查询条件是否正确
- 联系管理员确认权限设置

**Q: 统计报表数据不准确？**

A: 请确认：
- 选择的时间范围是否正确
- 统计维度和周期是否符合预期
- 联系管理员检查数据完整性

### 6.4 系统使用

**Q: 系统支持哪些浏览器？**

A: 推荐使用以下浏览器的最新版本：
- Chrome 90+
- Edge 90+
- Firefox 88+
- Safari 14+

**Q: 如何导出数据？**

A: 在统计报表页面，查询后可导出 Excel 或 PDF 格式的报表。

**Q: 系统支持移动端访问吗？**

A: 支持，系统采用响应式设计，可在手机浏览器中正常使用。
